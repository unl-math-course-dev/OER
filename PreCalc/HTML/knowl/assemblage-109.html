Properties of Equality
Addition:

If \(a = b\) and \(c\) is any number, then \(a + c = b + c\text{.}\)

Subtraction:

If \(a = b\) and \(c\) is any number, then \(a - c = b - c\text{.}\)

Multiplication

If \(a = b\) and \(c\) is any number, then \(a\cdot c = b\cdot c\text{.}\)

Division

If \(a = b\) and \(c\) is any nonzero number, then \(\frac{a}{c} =\frac{b}{c} \text{.}\)
