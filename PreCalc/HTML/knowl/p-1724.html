A graph does not change if we change the names of the variables, so we can let \(x\) represent the input for both functions, and let \(y\) represent the output. Consider the function \(C = h(F)\) from Example 158, and its inverse function, \(F = h^{-1}(C)\text{.}\) The formulas for these functions are \begin{align*} C \amp = h(F) = \frac{5}{9}(F - 32)\\ F \amp = h^{-1}(C) = 32 + \frac{9}{5}C. \end{align*} But their graphs are the same if we write them as \begin{align*} y \amp = h(x) =\frac{5}{9}(x - 32)\\ y \amp= h^{-1}(x) = 32 + \frac{9}{5}x. \end{align*} The graphs are shown in Figure 202.
