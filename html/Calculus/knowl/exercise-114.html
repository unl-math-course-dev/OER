Exercise9

Let \(u(x)\) be a differentiable function. For each of the following functions, determine the derivative. Each response will involve \(u\) and/or \(u'\text{.}\)

  1. \(p(x) = e^{u(x)}\)

  2. \(q(x) = u(e^x)\)

  3. \(r(x) = \cot(u(x))\)

  4. \(s(x) = u(\cot(x))\)

  5. \(a(x) = u(x^4)\)

  6. \(b(x) = u^4(x)\)
