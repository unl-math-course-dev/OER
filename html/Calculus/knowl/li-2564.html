Item4.3.9.d

We know that \(\int_0^4 c \, dx\) is the signed area of a rectangle with ``height" \(c\) (note that \(c\) could be negative), and base \(4\text{,}\) so

\begin{equation*} \int_0^4 c \, dx = 4c\text{.} \end{equation*}

Also, from part (c) we have

\begin{equation*} \int_0^4 [f(x) + g(x)] \, dx = \int_0^4 f(x) \, dx + \int_0^4 g(x) \, dx = -2+\frac{1}{2} + \frac{3\pi}{4} = -\frac{3}{2} + \frac{3\pi}{4}\text{.} \end{equation*}

Therefore,

\begin{equation*} \int_0^4 c \, dx = \int_0^4 [f(x) + g(x)] \, dx \end{equation*}

from which it follows that \(4c = -\frac{3}{2} + \frac{3\pi}{4}\) so \(c = -\frac{3}{8} + \frac{3\pi}{16}\text{.}\)
