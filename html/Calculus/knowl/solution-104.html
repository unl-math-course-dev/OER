Solution2.6.12.1

  1. The function \(h\) is invertible because its graph passes the horizontal line test.

  2. It does not seem possible to solve the equation \(y = x + \sin(x)\) for \(x\) in terms of \(y\text{.}\)

  3. By the rule for the derivative of an invertible function, we have \((h^{-1})'(\frac{\pi}{2} + 1) = \frac{1}{h'\left(\frac{\pi}{2} \right)}\text{.}\) Thus, we first compute \(h'(x)\) and find that \(h'(x) = 1 + \cos(x)\text{.}\) Since \(h'\left(\frac{\pi}{2} \right) = 1 + \cos\left(\frac{\pi}{2} \right) = 1\text{,}\) it follows that \((h^{-1})'(\frac{\pi}{2} + 1) = \frac{1}{1} = 1\text{.}\)
