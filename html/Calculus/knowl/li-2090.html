Item3.3.4.a

For \(s(t) = 3\sin(2(t-\frac{\pi}{6})) + 5\text{,}\) \(s'(t) = 3\cos(2(t-\frac{\pi}{6})) \cdot 2 = 6\cos(2(t-\frac{\pi}{6}))\text{.}\) It follows that \(s'(t) = 0\) if and only if \(\cos(2(t-\frac{\pi}{6})) = 0\text{,}\) and thus \(2(t-\frac{\pi}{6}) = \frac{\pi}{2} \pm k\pi\) for some integer value of \(k\text{.}\) Solving this most recent equation for \(t\text{,}\) we have \(t-\frac{\pi}{6} = \frac{\pi}{4} \pm \frac{k\pi}{2}\text{,}\) so

\begin{equation*} t = \frac{\pi}{6} + \frac{\pi}{4} \pm \frac{k\pi}{2} = \frac{5\pi}{12} \pm \frac{k\pi}{2}\text{.} \end{equation*}

Now we consider the given interval \([\frac{\pi}{6}, \frac{7\pi}{6}]\text{,}\) which we can think of as \([\frac{2\pi}{12}, \frac{14\pi}{12}]\text{.}\) The value \(t = \frac{5\pi}{12}\) lies in this interval, as does \(t = \frac{5\pi}{12} + \frac{\pi}{2} = \frac{11\pi}{12}\text{.}\) Thus, we have two critical numbers in the interval at which to evaluate \(s\text{,}\) as well as at the endpoints.

Doing so, we find:

  • \(s(\frac{5\pi}{12}) = 8\text{;}\)

  • \(s(\frac{11\pi}{12}) = 2\text{;}\)

  • \(s(\frac{\pi}{6}) = 5\text{;}\) and

  • \(s(\frac{7\pi}{6}) = 5\text{.}\)

The global max of \(s\) on \([\frac{\pi}{6}, \frac{7\pi}{6}]\) is thus \(s(\frac{5\pi}{12}) = 8\text{,}\) while the global min is \(s(\frac{11\pi}{12}) = 2\text{.}\)
