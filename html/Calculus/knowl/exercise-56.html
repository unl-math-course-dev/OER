Consider the graph of the function \(y = p(x)\) that is provided in Figure 1.69. Assume that each portion of the graph of \(p\) is a straight line, as pictured.

  1. State all values of \(a\) for which \(\lim_{x \to a} p(x)\) does not exist.

  2. State all values of \(a\) for which \(p\) is not continuous at \(a\text{.}\)

  3. State all values of \(a\) for which \(p\) is not differentiable at \(x = a\text{.}\)

  4. On the axes provided in Figure 1.69, sketch an accurate graph of \(y = p'(x)\text{.}\)
