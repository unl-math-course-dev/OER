We first note that \(p'(x) = 3x^2 - a\text{,}\) so to find critical numbers we set \(p'(x) = 0\) and solve for \(x\text{.}\) This leads to the equation \(3x^2 - a = 0\text{,}\) which implies

\begin{equation*} x^2 = \frac{a}{3}\text{.} \end{equation*}

If \(a \gt 0\text{,}\) then the solutions to this equation are \(x = \pm \sqrt{\frac{a}{3}}\text{;}\) if \(a \lt 0\text{,}\) then the equation has no solution. Hence, \(p\) has two critical numbers (\(x = \pm \sqrt{\frac{a}{3}}\)) whenever \(a \gt 0\) and no critical numbers when \(a \lt 0\text{.}\)
