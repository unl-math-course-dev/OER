It's not possible to say without more information: for instance, the functions \(f(x) = e^{-x}\text{,}\) \(f(x) = e^{-x}\text{,}\) and \(f(x) = x^2\) are each always concave up, but only one of them has a global minimum. We can say for sure that by being always concave up, \(f\) cannot have a global maximum, and it also cannot have both a global maximum and a global minimum. It might have a global minimum, or it might have neither.
