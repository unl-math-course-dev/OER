Item3.2.4.c

Using \(q(x) = \frac{1}{(x-c)e^x}\text{,}\) the quotient and product rules along with algebraic simplification show that

\begin{equation*} q'(x) = -\frac{(x-c)e^x + e^x}{(x-c)^2\left(e^x\right)^2} = -\frac{x-c+1}{(x-c)^2e^x}\text{.} \end{equation*}

The function \(q\) has critical points in its domain where \(q'(x) = 0\) or \(q'(x)\) doesn't exist. Note that \(q'(x)\) is undefined at \(x=c\text{,}\) but \(c\) is not in the domain of mq. Thus, the only critical points of \(q\) occur when \(q'(x)=0\text{.}\) The only way the fraction \(q'(x)\) can be \(0\) is if the numerator is \(0\text{.}\) Since \(x-c+1=0\) when \(x=c-1\text{,}\) the only critical number for \(q\) is \(x=c-1\text{.}\)
