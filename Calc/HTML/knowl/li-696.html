True: if a function \(p\) is differentiable at \(x = b\text{,}\) then \(\lim_{x \to b} p(x)\) must exist. This is true because we know that if \(p\) is differentiable at a point, then \(p\) is continuous there, and anytime a function is continuous at a point, it must have a limit there.
