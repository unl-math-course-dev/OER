The function is being evaluated at \(x_1 = 1.4\text{,}\) \(x_2 = 1.8\text{,}\) \(x_3 = 2.2\text{,}\) \(x_4 = 2.6\text{,}\) and \(x_5 = 3.0\text{.}\) Thus, since we know this is a right Riemann sum, we are using \(5\) subintervals with \(\triangle x = 0.4\text{,}\) and \(x_0 = 1.0\text{.}\) From the pattern, we observe that the function must be \(f(x) = x^2 + 1\) on the interval \([1,3]\text{.}\)
