The exact solution to the initial value problem \(\frac{dy}{dt} = 2t-1, \ y(0) = 0\) is \(y = t^2 - t\text{,}\) which we find by integrating both sides with respect to \(t\) and then applying the fact that \(y(0) = 0\) to determine that the integration constant is \(0\text{.}\) From here, we can see that the errors are \(e_1 = y(0.2) - y_1 = 0.04\text{,}\) \(e_2 = y(0.4) - y_2 = 0.08\text{,}\) \(e_3 = y(0.6) - y_3 = 0.12\text{,}\) \(e_4 = y(0.8) - y_4 = 0.16\text{,}\) \(e_5 = y(1.0) - y_5 = 0.2\text{.}\)
