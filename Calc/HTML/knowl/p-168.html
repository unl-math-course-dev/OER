  1. Observe that \(AV_{[3, 3+h]} = \frac{s(3+h)-s(3)}{h} = \frac{(3+h)^2 - 3^2}{h} = \frac{9 + 6h + h^2 - 9}{h} = \frac{6h + h^2}{h} = \frac{h(6 + h)}{h} = 6 + h\text{.}\)
  2. Using the expression just found in (a) with \(h = 0.2\text{,}\) \(AV_{[3,3.2]} = 6 + 0.2 = 6.2\) meters/min.
  3. Taking the limit of average velocity and using our work from (a), we find that
    \begin{equation*} IV_{t = 3} = \lim_{h \to 0} AV_{[3, 3+h]} = \lim_{h \to 0} 6+h = 6\text{,} \end{equation*}
    so the instantaneous velocity of the object when \(t = 3\) is 6 meters per minute.
