  1. \(h(x) = \sec(x)\) is defined for all \(x\) for which \(\cos(x) \ne 0\text{.}\) Hence the domain of \(h\) is all real numbers \(x\) such that \(x \ne \frac{k\pi}{2}\text{,}\) where \(k = \pm 1, \pm 2, \ldots\text{.}\)

  2. By the quotient rule,

    \begin{equation*} h'(x) = \frac{0 - 1 (-\sin(x))}{\cos^2(x)} = \frac{\sin(x)}{\cos^2(x)}\text{.} \end{equation*}

  3. Observe that \(h'(x) = \frac{\sin(x)}{\cos^2(x)} = \frac{1}{\cos(x)} \cdot \frac{\sin(x)}{\cos(x)}\text{,}\) so

    \begin{equation*} h'(x) = \sec(x) \tan(x)\text{.} \end{equation*}

  4. The derivative \(h'(x)\) is, like \(h(x)\text{,}\) defined for all values of \(x\) for which \(\cos(x) \ne 0\text{.}\) Therefore, \(h\) and \(h'\) have the same domain: all real numbers \(x\) such that \(x \ne \frac{k\pi}{2}\text{,}\) where \(k = \pm 1, \pm 2, \ldots\text{.}\)
