Solution5.3.9.1

  1. If we let \(u = \sin(x)\text{,}\) then \(du = \cos(x) \, dx\text{.}\) Since there is no cosine function present in the integrand, we don't have the needed function-derivative pair present in order to transform the integral into one we can evaluate.

  2. Using the Fundamental Trigonometric Identity, we observe that \(\sin^2(x) = 1-\cos^2(x)\text{,}\) and thus

    \begin{equation*} \sin^3(x) = \sin(x) \cdot \sin^2(x) = \sin(x) (1-\cos^2(x))\text{.} \end{equation*}

  3. By our work in (b), we see that \(\int \sin^3(x) \, dx = \int \sin(x) (1-\cos^2(x)) \, dx\text{,}\) and in this integral we have the function-derivative pair that corresponds to \(u = \cos(x)\) and \(du = -\sin(x) \, dx\text{.}\)

  4. Using this substitution and evaluating the resulting integral,

    \begin{align*} \int \sin^3(x) \, dx &= \int \sin(x) (1-\cos^2(x)) \, dx\\ &= \int (-1)(1-u^2) \, du\\ &= \int (u^2-1) \, du\\ &= \frac{1}{3}u^3 - u + C\\ &= \frac{1}{3}\cos^3(x) - \cos(x) + C\text{.} \end{align*}

  5. Here we observe that \(\cos^3(x) = \cos(x) \cdot \cos^2(x) = \cos(x) (1-\sin^2(x))\text{,}\) so letting \(u = \sin(x)\) and thus \(du = \cos(x) \, dx\text{,}\) we have

    \begin{align*} \int \cos^3(x) \, dx &= \int \cos(x) (1-\sin^2(x)) \, dx\\ &= \int (1-u^2) \, du\\ &= u - \frac{1}{3}u^3 + C\\ &= \sin(x) - \frac{1}{3}\sin^3(x) + C\text{.} \end{align*}
